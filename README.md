# Regicide.py
Solo only command-line implementation for boardgame Regicide. Runs anywhere python3 runs.

## Get Started
Almost no dependencies, simply run `python3 Regicide.py`.

## How-to-play
[CLICK HERE](https://www.regicidegame.com/site_files/33132/upload_files/RegicideRulesA4.pdf) for the game rules.  
In game, uses `?` to list commands.

## But Why Not A Deck Of Cards?
Written because I was bored at work and want a game to play in office without others noticing.  
I play it in VSCode terminal, it is quite stealth.
Didn't play for longer than the time it take to write it, but the coding itself is already interesting.

## Differences From official Digital Implementation(Regicide Companion)
There is no yield for regicide companion solo.
There is one here.
Also during implement I noticed a slight issue in the rules(which I thought was an oversight, after reaching out to Bad Prod they said it wasn't but the rules are still different from regicide companion) and I added a secret ending based on it.
